# Protocolo de Validação — Arquitetura em Camadas (SLF)
**Regra:** a arquitetura só vira padrão oficial se aprovada em 3 perfis: **Kid, Teen e Adult**.
**Como usar:** ao fim de cada aula, responda Sim/Não (2 minutos, de cabeça, sem reler a aula). Sim = bom. Marque e some.

---

## Checklist pós-aula (25 itens)

### A. Fluxo da aula
| # | Pergunta | S/N |
|---|---|---|
| A1 | Conduzi a aula inteira só com o que estava na tela (sem improvisar material)? | |
| A2 | As transições entre seções foram naturais (sem "peraí, deixa eu achar")? | |
| A3 | Os portões (CCQs 4/5, prática 80%) caíram no momento certo da aula? | |
| A4 | A ordem das seções funcionou sem eu precisar reorganizar? | |
| A5 | Usei todas as seções da Camada 1 (nenhuma sobrou inútil)? | |

### B. Qualidade pedagógica
| # | Pergunta | S/N |
|---|---|---|
| B1 | O aluno percebeu o padrão sozinho no Notice (antes da regra)? | |
| B2 | As CCQs revelaram algo sobre a compreensão que eu não teria visto? | |
| B3 | Pelo menos um erro do "Watch out" apareceu de verdade na aula? | |
| B4 | O aluno usou a estrutura espontaneamente na Task (sem eu pedir)? | |
| B5 | O resultado do Exit Ticket bateu com a minha percepção da aula? | |

### C. Experiência da professora
| # | Pergunta | S/N |
|---|---|---|
| C1 | O Teacher Help estava à mão quando precisei (sem procurar)? | |
| C2 | Os hints e follow-ups serviram como estavam escritos? | |
| C3 | Terminei a aula sem ter criado conteúdo de improviso? | |
| C4 | As camadas 3/4 ficaram fora do caminho até (se) eu precisar? | |
| C5 | Conduzi com menos esforço mental que uma aula preparada por mim? | |

### D. Experiência do aluno
| # | Pergunta | S/N |
|---|---|---|
| D1 | O aluno falou mais que eu na aula como um todo? | |
| D2 | Ele se manteve engajado até o Exit Ticket? | |
| D3 | A aula fluiu sem nenhum momento de frustração/travamento dele? | |
| D4 | Ele reagiu à personalização (sorriu/comentou quando viu o mundo dele nos exemplos)? | |
| D5 | No fim, ele soube dizer o que aprendeu hoje? | |

### E. Gestão do tempo
| # | Pergunta | S/N |
|---|---|---|
| E1 | A aula coube no horário sem cortar a Task ou o Exit Ticket? | |
| E2 | O plano de tempo do Teacher Help ficou perto do real (±20%)? | |
| E3 | Nenhuma seção ficou longa demais (aluno esperando)? | |
| E4 | Nenhuma seção ficou curta demais (faltou substância)? | |
| E5 | Se precisei de Camada 3 ou 4, a necessidade era real (não defeito da Camada 1)? *(se não usou, marque Sim)* | |

**Observações rápidas (1 linha cada, opcional):** o que travou · o que sobrou · o que faltou · frase do aluno que resume a aula.

---

## Critérios de aprovação

- **Por aula:** ≥20/25 "Sim" **e** nenhum aspecto com menos de 3/5.
- **Reprovação automática (independe da soma):** D1 = Não (aluno falou menos que a professora) ou E1 = Não (Task/Exit Ticket cortados).
- **Da arquitetura:** aprovada nos 3 perfis. Se falhar em 1 perfil, ajustamos a adaptação daquele perfil e retestamos só ele.
